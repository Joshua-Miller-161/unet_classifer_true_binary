# mlde_Josh_zarr_NB

When training the model for the first time on a new dataset, it is necessary to fit transformations to the data to prepare it for training. This will occur automatically provided no transforms already exist in AHHHHHH
 - Note: This process of fitting transforms is a little unstable on multiple GPUs, so I have used 1GPU when training the model for the first time on a new dataset, and then once the transforms are fitted one can train using as many GPUs as you want.
